### ✅ 2. 히어로 섹션 버튼 작동
히어로 섹션의 3개 버튼이 모두 정상 작동합니다:
- **상담·견적 요청** → `#rapid-quote` 섹션으로 스크롤
- **회사 소개 보기** → `#company-intro` 섹션으로 스크롤  
- **산업별 솔루션** → `#industry-playbooks` 섹션으로 스크롤

### ✅ 3. 폼 제출 데이터 확인 방법

#### 관리자 대시보드 접속
1. 브라우저에서 `admin/login.html` 접속
2. 로그인 정보:
   - **Username**: `admin`
   - **Password**: `admin123`

#### 리드 데이터 확인
1. 로그인 후 왼쪽 사이드바에서 **"리드 관리"** 클릭
2. 다음 폼 데이터를 확인할 수 있습니다:
   - 상담·견적 요청 (`intent: quote`)
   - 자료 다운로드 (`intent: resource`)
   - 15분 기술상담 (`intent: tech-call`)

#### 데이터 저장 위치
- 모든 폼 데이터는 `localStorage`의 `leads` 키에 저장됩니다
- 브라우저 개발자 도구 → Application → Local Storage에서도 확인 가능

## 🔧 기술 사양

### 폼 데이터 구조
```javascript
{
  id: "unique-id",
  intent: "quote|resource|tech-call",
  timestamp: "2026-01-20T...",
  company: "회사명",
  name: "담당자명",
  phone: "연락처",
  email: "이메일",
  industry: "업종",
  uph: "목표 UPH",
  solution: "관심 솔루션",
  preferredDate: "희망 일자",
  issue: "현재 이슈"
}
```

### localStorage 키
- `leads`: 모든 리드 데이터 배열
- `adminAuth`: 관리자 인증 상태
- `currentUser`: 현재 로그인한 사용자 정보

## 📞 문의 처리 프로세스

### 1. 상담·견적 요청
1. 사용자가 폼 작성 → localStorage에 저장
2. 관리자 대시보드에서 확인
3. 평균 4시간 이내 연락

### 2. 자료 다운로드
1. 사용자가 이메일 입력 → localStorage에 저장
2. 관리자가 확인 후 자료 이메일 발송

### 3. 15분 기술상담
1. 사용자가 희망 일자 선택 → localStorage에 저장
2. 담당 엔지니어 배정
3. 예약된 시간에 상담 진행
